# 自定义

本章介绍的能力（模板引擎、外部插件、404 页面等实际示例）同时适用于 Book 模式与 Site 模式，它们是站点呈现与内容处理层面的通用能力，不专属于任何一种使用方式。唯一的例外是 MoonBit 模块插件，由于它依赖直接编写 MoonBit 代码，仅适用于 Site 模式，相关页面开头也会重复这一点提醒。

## 各小节内容

**模板引擎**——模板语法、渲染上下文、严格与宽松两种渲染模式，请阅读 [模板引擎](./template.md)。

**自定义 404 页面**——一个综合示例，展示 Book 模式与 Site 模式各自如何处理同一类需求，请阅读 [自定义 404 页面](./custom-404.md)。

**插件系统**——三种扩展内容处理与呈现方式的途径：外部程序、Web Components、MoonBit 模块，请阅读 [插件系统](./plugins/index.md)。

Glob 模式匹配语法（用于静态资源字段与 Site 模式的 `Rule::Glob`）收录于 [Glob Patterns](../references/glob.md)，作为独立于具体使用场景的语法速查内容。
