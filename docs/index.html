MoPress 是一个现代文档与通用静态站点生成器，兼具 Book 模式的开箱即用生成文档，与 Site 模式的函数式可组合灵活性自定义原子构建生成任意静态站点。

快速开始 → 了解更多 
///|
async fn main {
  @core.mopress([
    Glob(
      "**/*.md",
      Text(raw => {
        raw
        |> @core.set_extension(".html")
        |> @core.render_markdown_and_frontmatter
        |> @core.load_and_apply_template("templates/default.html")
        |> @core.use_js(["console.log('hi, moonbit! hi, mopress!')"])
        |> @core.unify
      }),
    ),
    Glob("styles/**/*", Copy),
  ])
}

两种模式，一套底层

Book 模式

零代码，配置驱动。一份配置文件加一批 Markdown，即可构建完整站点。

Site 模式

用可组合的代码声明处理规则与渲染管线，设计思路继承自 Hakyll。

函数式

函数式编程，声明式内嵌 DSL 配置原子化构建流程

原子化

原子化构建，按需组合，灵活可扩展。

多种插件系统

外部程序、Web Components、MoonBit 模块，按需扩展，互不排斥。

原生 MoonBit

整个工具链用 MoonBit 编写，与生态原生契合。
