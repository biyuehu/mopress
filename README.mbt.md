# himeno/mobook

MoBook is an experimental MoonBit book generator inspired by mdBook. The goal is
to build a reliable MoonBit-native documentation tool rather than a direct port
of the Rust implementation.

The project is still in its foundation stage. Current work focuses on stable
configuration parsing, core book data structures, and the package boundaries
needed for summary parsing, Markdown handling, and rendering.

## Development

```sh
moon info
moon fmt
moon test
```
